# Architecture Decision Records

This directory contains ADRs for the Orkaive project — short records of
architectural decisions, the context that drove them, and the
consequences (good and bad) of each choice.

Use the [Nygard template](https://github.com/joelparkerhenderson/architecture_decision_records):

1. **Title** — a short noun phrase
2. **Status** — proposed / accepted / superseded
3. **Context** — the forces at play
4. **Decision** — what we chose
5. **Consequences** — what becomes easier, what becomes harder
